## 使用方法

1. 编辑 `jiankong_list.txt`，添加需要监控的端口
   - 格式：`IP 端口`
   - 端口仅支持百兆、千兆、万兆、四万兆，其他逻辑端口需要改 `core.formattext.ifname()`
   - 端口用简写，例 `E0/0/0`，需要大写
2. 执行 `initport.py`，预收集交换机所有接口信息，存储到 `data\hosts\`
3. 执行 `monitor.py`，开始监控
